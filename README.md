# TowerBattle

ブラウザ上で遊べる、オンラインマルチプレイ対応のどうぶつタワーバトル風ゲームです。

## 特徴
- **リアルタイム物理演算**: Matter.jsを利用し、ネットワーク越しに相手とリアルタイムでタワーバトルが楽しめます。
- **ピクセルパーフェクトな当たり判定**: 単純な多角形ではなく、画像の不透明なピクセル部分を自動解析して精密な当たり判定を生成するため、複雑な形状の動物同士が自然に重なり合います。
- **ターン制＆タイマー**: ターンごとに5秒のカウントダウンが発生し、相手とスムーズにゲームが進行します。
- **独自ステージ作成**: ブラウザ上でプラットフォームを配置し、オリジナルのステージ（部屋）を作成してオンライン対戦が可能です。

## 技術スタック
- フロントエンド: HTML5 Canvas, Matter.js, Socket.IO Client
- バックエンド: Node.js, Socket.IO

## 動かし方
1. `npm install` で依存関係（Socket.IO等）をインストールします。
2. `node server.js` でサーバーを起動します。（デフォルトポート: 3008）
3. ブラウザで `index.html`（サーバーと同じホスト）にアクセスしてプレイします。
