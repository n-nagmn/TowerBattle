# タワーバトル (Tower Battle)

Webブラウザ上で動作する、物理演算を活用した対戦型タワー積み上げゲームです。

## 特徴
- **マルチプレイ対応**: サーバーを介したロビーシステムで、他のプレイヤーとリアルタイム対戦が可能（ポーリング通信）。
- **リアルな物理演算**: `Matter.js` を使用し、動物たちの重さや摩擦、滑り落ちる挙動を精緻にシミュレーション。
- **完全同期**: クライアント間での物理エンジンのズレを防ぐため、最新の座標・角度をサーバー経由でスナップ同期。
- **ステージエディタ**: 右クリック・ドラッグ＆ドロップでオリジナルの足場を作成し、そのステージで募集をかけられます。
- **レスポンシブデザイン**: PCでもスマートフォンでも、画面サイズに合わせてキャンバスが見切れずに調整されます。

## 遊び方
ブラウザでアクセスするとロビー画面が表示されます。
1. **ルームに参加**または**ステージを作って対戦募集**を選択します。
2. 自分のターンが来たら、マウスを動かして落とす位置（X座標）を決めます。
3. **右クリック**: 落下させる動物を45度回転させます。（キーボードの `R` キーでも可）
4. **左クリック**: 動物を落下させます。（キーボードの `Space` キーでも可）
5. 盤面の外（下部）へ動物を落としてしまったプレイヤーの負けです。

## 動作環境・セットアップ
- PHP 7.4以上
- MySQL (MariaDB)
- PDO拡張モジュール

1. サーバーのWebルート（`/var/www/html/` 等）に本リポジトリのPHP/HTMLファイルを配置します。
2. MySQLデータベースを作成し、`database.sql` をインポートしてテーブルを構築します。
3. `db_connect.php` を開き、自身のデータベースのホスト、ユーザー名、パスワード、DB名に変更してください。
4. ブラウザから `index.html` にアクセスしてプレイ開始です。

## 使用ライブラリ
- [Matter.js](https://brm.io/matter-js/) (2D Physics Engine)
- [poly-decomp.js](https://github.com/schteppe/poly-decomp.js/) (Concave Polygon Decomposition)
